# 🏦 Bank System Management - C++ OOP

> A comprehensive console-based Bank Management System built with C++ and Object-Oriented Programming principles to manage clients, transactions, and user permissions.

## 🚀 Features
- Login System: Secure access with transaction logging.
- Client Management: Add, update, delete, and find bank clients.
- Transactions: Deposit, withdraw, and view total balances.
- User Permissions: Manage system users with specific access rights (Admin vs. Regular User).

## 🛠️ Technologies & Concepts
- Language: C++
- Paradigm: Object-Oriented Programming (Encapsulation, Inheritance, Polymorphism)
- Data Storage: File Handling (Text files for database simulation)
